Key Features

  • Tor Integration: Every command execution is routed through the Tor network to preserve anonymity.
  • Command Toralization: Use the toralize keyword to convert standard commands into Tor-protected executions.
  • Geolocation Masking: Hide your real IP and location from external services and observers.
  • Modular Architecture: Built with clean separation using core files like toralize.c, toralize.h, and Makefile.
  • Open-Source Community: Contributions are welcome from anyone passionate about privacy and Linux tooling.
