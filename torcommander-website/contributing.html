Contributing to TorCommander

We welcome all contributions from developers, security researchers, open-source enthusiasts, and privacy advocates.

How to Contribute

  1. Fork the TorCommander GitHub Repository.
  2. Create a new branch for your feature or fix.
  3. Write clean, well-documented code and commit your changes.
  4. Submit a pull request with a clear description of your update.

Contribution Areas

  • Enhancing anonymity features
  • Improving modularity and code structure
  • Adding new command wrappers
  • Fixing bugs or issues
  • Writing documentation
  • Design/UI improvements (like this website!)

We look forward to your contributions and ideas!
