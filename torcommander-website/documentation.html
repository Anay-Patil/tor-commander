Documentation

This section provides basic usage instructions for TorCommander.

Start the Tor Service

sudo service tor start

Run TorCommander

./torcommander

Sample Commands

Once TorCommander is running, you can execute commands through the Tor network like this:

curl --socks5-hostname 127.0.0.1:9050 https://check.torproject.org
