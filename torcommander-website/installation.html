Installation

To use TorCommander, make sure you have the following prerequisites installed on your Linux system:

  • Tor: Ensure the Tor service is installed and running.
  • GCC: Required for compiling the C program.
  • Make: Used to build the project using the Makefile.

Install Prerequisites

sudo apt update
sudo apt install tor gcc make

Clone the Repository

git clone https://github.com/Anay-Patil/tor-commander.git

Build the Project

cd tor-commander
make
